# Knowledge cheatsheets (reference data)

This directory holds India-specific reference data that the coach reads on demand. Currently empty (P0); will populate in P1.

Planned files:

- `ifct-cheatsheet.md` IFCT 2017 macros for staples (jowar, bajra, ragi, dals, paneer, oils)
- `icmr-dgi-2024.md` the 17 ICMR-NIN guidelines + My Plate
- `india-thresholds.md` Asian-Indian cutoffs (BMI ≥23, waist M≥90/F≥80, LAI 2024 LDL targets)
- `thin-fat-phenotype.md` why we use stricter cutoffs for South Asians
- `flexible-dieting.md` IIFYM evidence base (Conlin 2021, MASS 2025)
- `gi-india.md` glycemic index for common Indian foods (PMC9552392)
- `nafld-india.md` 38.6% adult prevalence; fructose load is primary lever
- `b12-d-deficiency.md` 51% / 80% prevalence in Indian vegetarians
- `vidarbha-cuisine.md` Saoji, varan-bhat, pithla, zunka, jowar bhakri, puran poli
- `workout-progression.md` beginner to intermediate gym path
- `india-festivals.md` Diwali, Holi, Ganesh, Navratri (vrat), wedding season
- `vrat-foods.md` sabudana, kuttu, singhara, samak rice, peanuts, potatoes
- `oura-recovery-interpretation.md` how to read Oura readiness/HRV/sleep
- `psychology-of-adherence.md` flexible vs rigid restraint
- `aqi-workout.md` indoor/outdoor decisions for Indian metros
- `indian-snack-culture.md` namkeen, chivda, biscuits, the background-calorie problem

These are framework content (educational, citation-backed, not personal). They live here in the framework so all vaults benefit from them.

In your personal vault, this directory is a **symlink** to here, so framework updates ship to your vault automatically.
